VHDL

VHDL is a Hardware Description Language that is used to describe systems such as integrated circuits and gate arrays. It is also used as a "parallel programming language." Parallel programming languages makes use of the "future concept". This means that one part of the code promises to give information to another part of the code at a future date or time rather than giving all the information at the current time.

VHDL is most often used to write code that describes a logic circuit. In programs such as VIVADO, the VHDL code is processed and synthesized before it can be simulated.
